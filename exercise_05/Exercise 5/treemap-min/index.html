treemap — Introduction to Data Visualization

Research branch of the treemap renderer of Seerene developed by the Computer Graphics Systems Group at the HPI.

WebGL About Page: webgl-operate.org/about.html  | WebGL Operate: webgl-operate.org

This map visualizes the bitcoint repository. The lines of code in all .cpp, .h and .py files are mapped to the base area of the cuboids. The height of the cuboids represents the number of commits on this file. The color reperesents the number of different contributors that have commited to this file. Therefore green stands for many different authors, while red stands for only a few authors.
